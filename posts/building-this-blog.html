我使用 ChatGPT 5.5 搭建了这个博客

这是我的第一篇博客文章，记录我使用 ChatGPT 5.5 搭建这个博客的经历，以及一些有趣的细节。

这是我用 ChatGPT 5.5 花了一天时间搭建出来的 Blog。这里是第一篇文章，我很高兴你能点进来看看。

使用到的工具有 ChatGPT 5.5、Codex、Cloudflare DNS、GitHub Pages。

费用统计

Cloudflare 域名费用：第一年约 7.5 USD。

让我们开始吧

Part 1：使用 ChatGPT 生成 HTML 页面代码

我在这里使用的是 ChatGPT 5.5，并且让 Codex 直接生成主页、目录页和文章页模板。之后我根据自己的需求继续调整样式、交互和页面结构。给我的感觉是，这个模型真的很强。

Part 2：购买 Cloudflare 域名

我这里使用的是 Cloudflare 的域名服务，也可以使用 Aliyun、Tencent、GoDaddy 等域名服务商。Cloudflare 相对来说比较便宜，我购买的顶级域名是 nyaro.org，第一年约 7.5 USD，第二年约 10 USD。

Part 3：获取 GitHub Pages URL 并上传代码

GitHub Pages 已经有很完整的官方文档，所以这里不展开重复说明。可以参考： GitHub Pages 快速入门

Part 4：配置 DNS 设置

购买域名后，需要在 Cloudflare DNS 中添加 GitHub Pages 要求的 DNS 记录。GitHub Pages 通常会给出四条 A 记录和一条 CNAME 记录。

CNAME 记录

将 www 子域名指向你的 GitHub Pages 地址，例如 username.github.io。

也可以参考 GitHub 官方文档： 配置 GitHub Pages 自定义域名

Part 5：GitHub Pages 自定义域名

配置 DNS 后，还需要在 GitHub Pages 的设置中添加你的自定义域名。进入仓库的 Settings > Pages，在 Custom domain 中输入域名并等待验证。

Part 6：大功告成

这个时候网站可能还没有完全生效。等待 10 分钟到 1 小时后，再访问你的域名或 www 子域名，通常就可以看到网站了。

结束

这样你就可以比较轻松地搭建一个属于自己的博客了。
